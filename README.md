# 项目介绍
    电控与视觉实现镖架制导的关键部分代码

# 前言
在机械保证单发镖体姿态稳定、落点一致的情况下，我们在算法设计上提出了以下构想：  
利用神经网络识别出基地靶上的绿灯，并在三分钟准备阶段时以识别出绿灯像素点的圆心作为坐标原点。  
在比赛过程中飞镖靶位绿灯移动时，视觉识别系统计算出当前绿灯位置与坐标原点的像素差值，  
并将该值发送给电控系统。电控系统利用PID算法以达到飞镖YAW轴跟随的效果。
